命令行 find 查找

  • 指定找文件
    find / -iname "*.pdf" -type f
  • 指定找文件夹
    find / -iname "*.pdf" -type d
  • 通用查找
    find . "*" | grep -P ".jpg"
    find / -iname "*.pdf"
  • 使用正则表达式查找,-E是扩展的正则查找
    find -E . -regex ".*.jpg"
