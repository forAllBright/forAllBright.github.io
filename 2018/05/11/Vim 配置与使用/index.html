Vim 安装时

Mac 环境下，Vim 在安装之前就应该想好要用到支持哪些语言的编译，否则在安装好后需要某个插件时但是之前没有支持编译再重新安装 Vim 会很麻烦。且对应的 python 的版本号要注意。

注意： Neocomplete 需要 lua , YouCompleteMe 需要 python 。
